Setup Instructions

Get started with these simple steps to run the project locally.

1

Download and Extract

Download the zip file and extract it to your desired location.

2

Install Dependencies

Run npm install to install all required packages.

3

Create Firebase Project

Create a new project at console.firebase.google.com

4

Enable Firebase Services

Enable Authentication (Email/Password), Firestore Database, and Storage.

5

Configure Firebase

Copy your Firebase config to src/services/firebase.js

6

Start Development Server

Run npm start and open http://localhost:3000
